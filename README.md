# cluster 会場コメント オーバーレイ

cluster のイベントページに表示される「会場のコメント」を、見やすいオーバーレイとして表示する Chrome 拡張機能です。

イベントページを開いたまま、コメント欄だけを画面上に重ねて表示できます。表示サイズ、更新間隔、色、背景を隠すレイヤーなどを調整できます。

## できること

- cluster のイベントページで「会場のコメント」をオーバーレイ表示
- コメントを一定間隔で自動更新
- オーバーレイの幅・高さ・更新間隔を変更
- 背景色、文字色、投稿者名の色を変更
- 色を直接入力、またはカラーパレットから選択
- 元ページを隠す背景レイヤーの ON/OFF と色変更
- オーバーレイの影の ON/OFF
- 投稿者アイコンをコメント一覧に表示
- コメント一覧のスクロールバーを非表示にしてすっきり表示

## 対応ブラウザ

Google Chrome を想定しています。

Microsoft Edge など Chromium 系ブラウザでも動く可能性がありますが、主な確認対象は Chrome です。

## インストール方法

この拡張機能は Chrome ウェブストアではなく、GitHub からダウンロードして手動で読み込む形式です。

1. GitHub のこのページを開きます。
2. 緑色の `Code` ボタンを押します。
3. `Download ZIP` を選んでダウンロードします。
4. ダウンロードした ZIP ファイルを展開します。
5. Chrome で `chrome://extensions/` を開きます。
6. 右上の `デベロッパー モード` を ON にします。
7. `パッケージ化されていない拡張機能を読み込む` を押します。
8. ZIP を展開してできたフォルダを選びます。

これで拡張機能が Chrome に追加されます。

## 使い方

1. Chrome で cluster にログインします。
2. cluster のイベントページを開きます。
3. ページ右下に「会場のコメント」オーバーレイが表示されます。
4. オーバーレイ上部のボタンや設定欄から表示を調整します。

イベントページをすでに開いている場合は、拡張機能を読み込んだあとにページを再読み込みしてください。

## オーバーレイ上の操作

- 歯車ボタン: 設定欄の表示・非表示
- 更新ボタン: コメントを手動更新
- `−` ボタン: オーバーレイを折りたたみ
- ヘッダー部分をドラッグ: オーバーレイを移動

## 設定できる項目

- 幅: オーバーレイの横幅
- 高さ: オーバーレイの高さ
- 秒: コメントを再読み込みする間隔
- 背景色: オーバーレイ本体の背景色
- 背景隠し色: 元ページを隠すレイヤーの色
- 文字色: コメント本文の文字色
- 投稿者名色: 投稿者名の表示色
- 背景を隠す: オーバーレイの後ろに単色レイヤーを表示
- 影を表示: オーバーレイ周りの影を表示

色は `#ffffff` のような形式で入力できます。`rgb(255, 255, 255)` 形式も使えます。カラーパレットから選ぶこともできます。

## 拡張機能アイコンからの設定

Chrome のツールバーにある拡張機能アイコンを押すと、ポップアップからも設定できます。

ポップアップには次のボタンがあります。

- 保存: 入力した設定を保存
- 更新: 開いている cluster ページのコメントを更新
- 表示切替: オーバーレイの表示・非表示を切り替え

## 応用例: OBS で配信・録画に取り込む

cluster イベントをプロカメラマンモードで配信・録画するときに、コメント表示用の画面として使うこともできます。

たとえば次のような設定にします。

- 背景を隠す: ON
- 影を表示: OFF
- 更新間隔: 短めの秒数
- 背景隠し色: OBS 側で抜きやすい色、または配信画面に合わせた単色

この状態の Chrome 画面を OBS の `ウィンドウキャプチャ` や `画面キャプチャ` で取り込み、OBS 側のフィルターで色抜き・クロップ・位置調整などを行うと、cluster の配信画面に会場コメントを重ねやすくなります。

背景を隠すレイヤーを使うと、元の cluster ページが映り込みにくくなります。影を非表示にすると、OBS 側でコメント欄だけを合成するときに輪郭が目立ちにくくなります。

## 表示されないとき

次の点を確認してください。

- cluster にログインしているか
- 開いているページが cluster のイベントページか
- 拡張機能を読み込んだあと、ページを再読み込みしたか
- Chrome の拡張機能一覧で、この拡張機能が有効になっているか
- イベントページ側に「会場のコメント」が存在するか

それでも表示されない場合は、拡張機能を一度削除してから、もう一度読み込んでください。

## 注意事項

この拡張機能は、ログイン済みの cluster ページ上で利用することを想定しています。

cluster の画面構成や API の仕様が変わった場合、コメントを取得できなくなることがあります。その場合は拡張機能側の更新が必要です。

この拡張機能は cluster の公式機能ではありません。
